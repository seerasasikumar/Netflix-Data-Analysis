# 🐍 Python Analysis

## 📌 Overview

This folder contains the Python-based analysis of the Netflix dataset using Pandas and Matplotlib. The objective is to clean the data, perform exploratory data analysis (EDA), and visualize key insights about Netflix Movies and TV Shows.

---

## 🛠️ Libraries Used

- Pandas
- Matplotlib

---

## 📊 Analysis Performed

### Data Exploration

- Loaded the Netflix dataset
- Displayed the first and last records
- Checked dataset shape
- Displayed column names
- Inspected data types and dataset information

### Data Cleaning

- Checked missing values
- Filled missing values in:
  - Director
  - Cast
  - Country
  - Rating
  - Date Added
  - Duration
- Checked duplicate records

### Exploratory Data Analysis (EDA)

- Movies vs TV Shows count
- Content Rating distribution
- Top 10 Countries by content
- Top 10 Genre categories
- Content Release Year trend
- Top 10 Directors

---

## 📈 Visualizations

The following charts were created using Matplotlib:

- Movies vs TV Shows (Bar Chart)
- Top 10 Netflix Content Ratings (Bar Chart)
- Top 10 Countries by Netflix Content (Bar Chart)
- Top 10 Genre Categories (Bar Chart)
- Netflix Content Release Over the Years (Line Chart)
- Top 10 Directors on Netflix (Bar Chart)

---

## 💡 Key Insights

- Movies are more numerous than TV Shows on Netflix.
- The United States contributes the highest number of Netflix titles.
- Drama and International content are among the most common genres.
- Netflix content has grown significantly over the years.
- A few directors have contributed multiple titles to the platform.

---

This Python analysis complements the Excel, SQL, and Power BI analyses to provide comprehensive insights into the Netflix dataset.
