# 📊 Excel Analysis

## 📌 Overview

This folder contains the Microsoft Excel analysis of the Netflix dataset.

The analysis focuses on understanding the distribution of Movies and TV Shows, release trends, content ratings, countries, and genres using Excel features.

---

## 📊 Analysis Performed

- Data Cleaning
- Data Formatting
- Pivot Tables
- Pivot Charts
- KPI Calculations
- Dashboard Creation

---

## 📈 Key Metrics

- Total Titles
- Total Movies
- Total TV Shows
- Total Countries
- Content Ratings
- Release Year Distribution

---

## 📊 Visualizations

- Movies vs TV Shows
- Titles by Rating
- Titles by Country
- Top Genres
- Release Year Trend

---

## 💡 Key Insights

- Movies make up the majority of Netflix content.
- Drama and International Movies are among the most common genres.
- The United States contributes the largest number of titles.
- Netflix content has grown significantly over recent years.
